## インストールが必要なソフトウェア

### 1. Python 3.8以上（必須）

MCPサーバーの実装に必要です。

#### インストール手順

1. **公式サイトにアクセス**
   - https://www.python.org/downloads/

2. **最新版をダウンロード**
   - 「Download Python 3.x.x」ボタンをクリック
   - 推奨：Python 3.12 以上

3. **インストール実行**
   - ダウンロードしたインストーラーを実行
   - **重要**: 「Add Python to PATH」に必ずチェックを入れる
   - 「Install Now」をクリック

4. **確認**
   - コマンドプロンプトを開く
   - 以下のコマンドを実行：
     ```
     python --version
     ```
   - Python 3.x.x と表示されればOK

---

### 2. SQLite（必須）

ローカルデータベースとして必要です。

#### インストール手順

1. **公式サイトにアクセス**
   - https://www.sqlite.org/download.html

2. **Windows用バイナリをダウンロード**
   - 「Precompiled Binaries for Windows」セクション
   - 「sqlite-tools-win-x64-*.zip」をダウンロード

3. **解凍と配置**
   - ダウンロードしたZIPファイルを解凍
   - 推奨場所：`C:\sqlite\`

4. **環境変数PATHに追加**
   - Windowsキー + R → 「sysdm.cpl」と入力 → Enter
   - 「詳細設定」タブ → 「環境変数」ボタン
   - 「システム環境変数」の「Path」を選択 → 「編集」
   - 「新規」をクリック → SQLiteのパス（例：`C:\sqlite`）を追加
   - 「OK」で全てのウィンドウを閉じる

5. **確認**
   - コマンドプロンプトを開く（新しいウィンドウ）
   - 以下のコマンドを実行：
     ```
     sqlite3 --version
     ```
   - バージョン情報が表示されればOK

---

## インストール完了後の確認方法

両方のインストールが完了したら、以下のコマンドを実行してください：

```
python --version
sqlite3 --version
```

両方のバージョン情報が表示されれば成功です。

---

## インストール完了後にお知らせください

インストールが完了したら、チャットで「インストール完了しました」とお知らせください。
PMが環境を確認し、Directorがシステム設計を開始します。

---

## トラブルシューティング

### Pythonが認識されない場合
- コマンドプロンプトを再起動してください
- PATHが正しく設定されているか確認してください

### SQLiteが認識されない場合
- コマンドプロンプトを再起動してください
- 環境変数PATHが正しく設定されているか確認してください
- パスの最後に `\` がないか確認してください

### それでも解決しない場合
具体的なエラーメッセージを教えてください。サポートします。

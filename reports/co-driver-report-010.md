## 使い方（簡単な手順）

### 1. データベースを準備

```bash
python src/database/init_db.py
```

### 2. Webサーバーを起動

```bash
cd src/web
node server.js
```

→ `http://localhost:3000` で起動

### 3. Webクライアントを起動

```bash
cd src/web/client
npm install
npm run dev
```

→ `http://localhost:5173` でアクセス可能

### 4. Claude Desktopでキャラクターを作る

**AI自動生成**:
```
create_character(
  account_id=1,
  name="炎の戦士",
  prompt="熱血漢で攻撃一辺倒。仲間思い。",
  auto_allocate=True,
  auto_select_abilities=True
)
```

→ プロンプトを書くだけで、AIが自動的にステータスとアビリティを決定

### 5. マッチングに参加してバトル

```
join_queue(character_id=1)
```

→ 自動的に相手を見つけてバトル開始

### 6. Webブラウザで観戦

`http://localhost:5173` でバトル一覧を見て、リアルタイムで観戦

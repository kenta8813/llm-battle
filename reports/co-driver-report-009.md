## 使い方

### 準備（初回のみ）

**1. データベースを準備する**
```
cd C:\Users\kenta\projects\llmbattle
python src/database/init_db.py
```
これで、データの保管庫が作られます。

**2. 必要なパッケージをインストールする**
```
cd C:\Users\kenta\projects\llmbattle\web-client
npm install
```
これで、ブラウザ画面に必要な部品が揃います。

### 起動方法

**3つのサーバーを起動します：**

**ターミナル1: MCPサーバー（Claudeの操作パネル）**
```
cd C:\Users\kenta\projects\llmbattle
python src/mcp-server/main.py
```

**ターミナル2: Webサーバー（情報の配達員）**
```
cd C:\Users\kenta\projects\llmbattle\web-server
node server.js
```
→ `http://localhost:3000` で起動します

**ターミナル3: Webクライアント（ブラウザ画面）**
```
cd C:\Users\kenta\projects\llmbattle\web-client
npm run dev
```
→ `http://localhost:5173` で起動します

**4. ブラウザでアクセスする**

ブラウザで以下にアクセスしてください：
```
http://localhost:5173
```

これで、ゲーム画面が見られます。

## 今できること

### Claudeができること（13種類の操作）

MCPサーバーを通じて、Claudeは以下を自由に操作できます：

1. アカウント管理
   - アカウントを作る
   - 自分のアカウント情報を見る

2. キャラクター管理
   - キャラクターを作る
   - キャラクターの情報を見る
   - キャラクターの詳細な統計を見る
   - キャラクターを削除する

3. バトル操作
   - バトルに参加する（マッチング待機）
   - 通常攻撃をする
   - スキルを使う
   - バトルから離脱する

4. 情報確認
   - ランキングを見る
   - 全体の統計を見る

### ブラウザで見られるもの

1. トップページ
   - すべてのアカウント一覧
   - 各アカウントの勝率・レーティング

2. キャラクター詳細ページ
   - キャラクターのステータス
   - 覚えているスキル
   - バトル履歴

3. バトル観戦ページ
   - リアルタイムでバトルが見られる
   - ターンごとの行動が自動更新される
   - ダメージやスキル効果が表示される

4. ランキングページ
   - 上位10キャラクターのランキング
   - 勝率やバトル数の統計

# LLM Battle Game バトル戦略ガイド

**バトル中の行動判断を深掘りするスキルです。play / quickmatch と組み合わせて使います。**

---

## 行動選択の優先順位

### 1. 人間の作戦を最優先

バトル開始前に伝えられた作戦を常に意識し、それに沿った行動を選ぶ。
作戦がない（お任せ）場合は以下の基準で判断。

---

### 2. HP状況による緊急判断

| 自分のHP残量 | 行動 |
|-------------|------|
| 40%以下 | heal ability を即使用。CDなら defend で延命 |
| 20%以下 | defend または dodge で延命を最優先 |
| 十分（60%以上） | 攻撃的な行動を選ぶ |

---

### 3. ステータス差を活かす

**自分のATKが相手DEFより大幅に高い場合**
→ attack / ダメージ系ability を積極的に使う。defend/dodgeは損失

**自分のSPDが相手より高い場合**
→ 先攻を活かして先に削る。dodge も成功率が高い

**自分のDEFが高い場合**
→ defend を挟みながら相手のリソースを削る持久戦が有効

**自分のSPDが相手より低い場合**
→ dodge の成功率が下がる。defend か attack を優先

---

### 4. アビリティの使いどころ

#### ダメージ系（damage）
- CDが0なら積極的に使う
- power=200以上の高威力技（必殺技など）は温存するか序盤から使うかは作戦次第
  - 「序盤全力」作戦 → 即使用
  - 「終盤温存」作戦 → 相手HP50%以下になってから

#### 回復系（heal）
- HP 40%以下になったら即使用
- CDが明けるまで待てるなら待つ。待てないなら defend で凌ぐ

#### バフ系（buff）
- 序盤（ターン1〜2）に使って有利状況を作るのが基本
- 効果は1ターン限りが多いため、攻撃と組み合わせて使う

#### デバフ系（debuff）
- 序盤に相手の高いATK/SPDを下げると以後の被ダメが減る
- ただし自分が先攻できる状況ならdebuffより攻撃の方が有利なことも多い

---

### 5. dodge を使うタイミング

- 自分SPD > 相手SPD → 成功率が高い（有効）
- 自分SPD < 相手SPD → 成功率が低い（基本的に使わない）
- 相手が高威力abilityを連発してきた場合の回避策として

---

## クールダウン管理の考え方

バトル中は各アビリティのCDを頭に入れておく。

```
例: 必殺技（CD=3）をターン1に使った場合
→ ターン2,3,4 は使えない
→ ターン5から再び使える
```

次の大技を使えるターンを逆算し、それまでの間を通常攻撃やバフ/デバフで埋める。

---

## ターンごとの報告フォーマット

各ターンを以下の形式で人間に報告：

```
【ターンN】
▶ 自分の行動: ○○（理由: ...）
─────────────
⚔ 与ダメージ: XX
🛡 被ダメージ: XX
✨ エフェクト: ...

残HP → 自分: XX/XX | 相手: XX/XX
```

## 作戦別の行動傾向

| 作戦 | 行動傾向 |
|------|---------|
| 序盤全力 | ターン1から最高火力のabilityを使い続ける |
| 慎重 | 相手の行動を見てから反応。HP管理を重視 |
| 必殺技温存 | 相手HP50%以下まで通常攻撃で削り、止めを必殺技で |
| 長期戦 | heal・defend を多用し、相手のリソース切れを狙う |
| お任せ | 上記のステータス差・HP状況の基準に従って最適行動を選ぶ |
